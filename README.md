# AQI
空气质量指数AQI查询软件

### 数据来源 
定位数据来自BaiduLBS:  
http://developer.baidu.com/map/wiki/index.php?title=android-locsdk/guide/v5-0  

AQI数据来自pm25.in:  
http://www.pm25.in/api_doc

###功能介绍：  
1） 根据用户网络确定用户所在城市  
2） 根据定位城市查询AQI数据
3） 除了支持默认定位查询之外，用户可以长按按钮，在城市列表中选择自己感兴趣的城市查询AQI数据  
4） 单击按钮显示最近三次查询的详细数据（包括AQI，以及主要污染物含量等信息）
